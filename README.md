# Phishing Email Detection Model

This project is a Machine Learning based phishing email detection system developed using Python and Scikit-learn. The model analyzes URLs and classifies them as either phishing or safe.

## Features

- Detects phishing and legitimate URLs
- Uses Machine Learning for classification
- Data preprocessing using CountVectorizer
- Accuracy evaluation
- Confusion matrix visualization using Seaborn and Matplotlib

## Technologies Used

- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn

## Dataset

Dataset used from Kaggle:
Phishing Site URLs Dataset

## Project Workflow

1. Load dataset
2. Preprocess URL data
3. Convert text into numerical vectors
4. Train Machine Learning model
5. Test model accuracy
6. Display confusion matrix

## Output

- Classification of URLs as phishing or safe
- Accuracy score
- Confusion matrix graph
